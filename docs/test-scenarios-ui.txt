BƯỚC 6: NHẬP INTENT VÀ SINH SPEC.md
--------------------------------------------------------------

Click tab "Documents"

Trong ô SPEC, nhập vào ô Intent:

  Xây dựng ứng dụng Todo đơn giản bằng Python và FastAPI.
  Cho phép tạo, xem, cập nhật, xoá task.
  Mỗi task có tiêu đề, mô tả, trạng thái (todo/done).
  Có API REST và file test pytest.

Nhấn "Generate SPEC"

Chờ 20-60 giây

PASS nếu:  SPEC.md hiện ra với badge "Draft"
           Nội dung có các phần:

             ## Goals
             ## Functional Requirements
             ## Acceptance Criteria

--------------------------------------------------------------
BƯỚC 10: CHẠY CODER AGENT
--------------------------------------------------------------

Vào tab Kanban
Thấy các task trong cột TODO

Kéo task đầu tiên (ví dụ "Thiết lập project FastAPI") sang cột IN PROGRESS
(hoặc click nút Start trên card)

Quan sát panel bên phải (Thought Stream):
  - Thấy: "Coder LLM round 1: invoking model..."
  - Thấy: TOOL_CALL → write_file: main.py
  - Thấy: TOOL_CALL → run_terminal: python -m pytest -q
  - Thấy: TOOL_RESULT: kết quả test

Chờ Agent chạy xong (5-10 phút tùy task)

PASS nếu:  Task tự chuyển sang cột REVIEW
           Panel hiển thị "STATUS_CHANGE: CODING → REVIEWING"
FAIL nếu:  Task về lại TODO với thông báo lỗi

BƯỚC 12: DUYỆT CODE (APPROVE)
--------------------------------------------------------------

  - Dòng xoá màu đỏ
  - AI Reviewer comment bên dưới (score, nhận xét)

Nhấn "Approve"

PASS nếu:  Task chuyển sang DONE
FAIL nếu:  Lỗi hoặc task không chuyển

--------------------------------------------------------------
BƯỚC 13: TỪ CHỐI CODE VÀ YÊU CẦU SỬA
--------------------------------------------------------------

Với task khác ở REVIEW

Trong ô feedback nhập:

  Hàm get_task thiếu kiểm tra task không tồn tại,
  cần trả về 404 thay vì để crash.

Nhấn "Reject"

PASS nếu:  Task về IN PROGRESS
           Agent chạy lại, stream hiển thị "PO review feedback: ..."
           Sau khi xong → REVIEW với code đã sửa
FAIL nếu:  Task không chuyển, hoặc Agent không nhận được feedback

--------------------------------------------------------------
